Providing data through service endpoints

In the third semester, I’ve chosen for the Web API Development elective. It has a duration of 6 weeks and is focused on server-side programming and web development. For our exam we develop a RESTful web API for a self-chosen problem.



CHALLENGE

The Harry Potter universe is huge and there are a lot of movies to watch to understand the whole story and to catch every detail. However, not everyone has 19 hours to have a marathon and most fandom resources online are so overwhelming that you don’t know where to even start.



SOLUTION

Our web API provides the most relevant data revolving Hogwarts in a way that is easy to digest. It has defined service endpoints that the user can access to get data from our relational database. It also handles authenticated and authorised access.

MY ROLES

UI Design, Front-end & Back-end Development



LOCATION

Scottish Highlands, Great Britain :)

View Prototype

CHANGING THE MINDSET

During this elective we were introduced to the Node.js run-time environment for using JavaScript in server-side programming. We also learned how to plan, design and implement relational databases suitable for storing and managing complex data sets.


I'd say changing my mindset was the most challenging. For example, I had to keep reminding myself that the structure of the data we request did not have to be identical to the database structure. It took me quite some time to get the hang of this.


Even though these 6 weeks were intense, I do not regret choosing this elective. I spent the majority of my time debugging and I learned a lot from that.

DISPLAYING JAVASCRIPT OBJECTS

What I also learned is how to create HTML elements in JavaScript and to display JavaScript objects.


All our school projects have to be uploaded to our student server. But due to security reasons the front-end could not be connected to the back-end, meaning the interface would not be functional.


My team and I however still wanted our website to be more interactive by showing data in one way or another. So I made a JavaScript object and created a function that would display that on click. I used elements in the JavaScript object to create HTML elements as well and I was so happy I knew how to. I remember how lost I felt during a lecture last year about exactly this, so this really felt like an accomplishment.
